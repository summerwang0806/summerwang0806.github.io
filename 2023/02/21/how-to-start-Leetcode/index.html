how to start Leetcode?

Created2023-02-21|Updated2023-02-21
|Post View:

本文记录Leetcode完整版实践过程，并且持续更新改进的方法，以及不定期更新一些c++知识

怎么开始刷题？

打开界面后，题目区域和答题区域一目了然。可以在网页中直接作答（更加便捷），也可以在自己的ide工具下答题（比较麻烦）。
如果在自己的工具下进行编写，不仅仅需要完成class的编写，还需要补充完整的程序，下面记录一下在ide下进行编写程序的方法（实际上就是怎样在自己的ide工具跑出c++程序，下面均在vscode下）。
可以使用简单的按钮一键编译运行，但是为了熟悉指令，选择使用powershell用命令行执行。

编译所需指令：g++ -o newname '.\test.cpp'
运行所需指令：.\newname.exe

其中 -o<文件> 的意思是将输出输入到指定文件。

其实可以简化一下，将两条指令写成一个脚本直接运行（就像计网实践时候以及大工程项目编写的时候）。如果脚本执行失败时，可能是因为默认不允许执行脚本，可以参考这篇博客修改。不过，还没弄明白……
测试时候更改输入输出即可。

hexo使用

  • 写心得博客
    （在D盘打开summerblog）在git-bash中输入hexo new 'BlogName'即可。
  • 上传
    生成静态文件：hexo generate或者hexo g
    预览启动本地端口：hexo server或者hexo s
    部署到github仓库：先hexo clean清空缓存，之后hexo deploy -g或者hexo d -g
  • 一些报错记录
    • 可能是github没连上
    • 可能仓库大小不够了，解决方法参考这篇博客
